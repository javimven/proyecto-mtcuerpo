-MTCorporal.apk es la aplicación para monitorizar los datos en android.

-completo.ino es el programa utilizado para el sistema completo (2 puentes + UV) [BT+microSD]

-diodo.ino es el programa utilizado para medir radiación UV.

-divisor.ino es el programa utilizado para medir temperatura con un divisor de tensión

-fotodiodo.ino es el programa utilizado para medir radiación solar.

-puenteW.ino es el programa utilizado para medir temperatura con 2 puentes 
de Wheatstone.

-puenteyampli.ino es el programa utilizado para medir temperatura con un puente conectado
a un amplificador de instrumentación.
